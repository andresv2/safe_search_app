== Sandbox

<em>Better Browsing for Kids</em>

This project is was created for General Assembly's cross-immersive hackathon. Sandbox will eventually be a Chrome extension that will provide children with a safe and personal internet experience.

See it here: https://safesandbox.herokuapp.com/
